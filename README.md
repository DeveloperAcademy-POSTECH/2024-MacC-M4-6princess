# 🎀 Frameet 🎀 언제 어디서나 최애와 함께 사진 찍기


## 🎯 Challenge Statement

아이돌을 좋아하는 팬들이 **좋아하는 연예인과 함께 찍은 것 같은 사진을 쉽게 만들 수 있도록 돕자.**

## 👥 Target Users

- 아이돌을 좋아하는 모든 팬  
- 특히 **포토부스에서 네 컷 사진을 찍어본 경험이 있는 팬**  

<!--


## 🔎 User Research

### 🔬 Interview 1: 팬 문화 전반 파악  
- **목표**: 팬들의 활동 흐름과 니즈 확인  
- **결과**: 앱을 통한 문제 해결이 유의미할 수 있다는 판단

## 💡 Solution Concept

> 팬이 아이돌 사진을 불러오고,  
> 자신이 원하는 사진을 넣어 **자동 배경 제거 + 프레임 조합**으로  
> **마치 아이돌과 함께 찍은 듯한 결과물**을 만들 수 있는 앱!



## 📱 핵심 기능 소개

| 순서 | 기능 |
|------|------|
| 1 | 다양한 아이돌 프레임 선택 |
| 2 | 사진 선택 & 배경 자동 제거 |
| 3 | 스티커, 텍스트로 꾸미기 |
| 4 | 사진 저장 및 공유 가능 |
| 5 | 무제한 촬영 & 집에서도 자유롭게 가능 |
